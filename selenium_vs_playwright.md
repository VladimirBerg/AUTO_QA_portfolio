# Playwright vs Selenium

| Критерий | Playwright | Selenium |
|----------|------------|----------|
| Установка | pip install + браузеры | pip install + webdriver |
| Скорость | Быстрее | Медленнее |
| Автоожидания | Встроены | WebDriverWait |
| Сетевой перехват | Из коробки | Через прокси |
| Мобильная эмуляция | Встроена | Appium |
| Документация | Отличная | Отличная |

## Вывод
Playwright — лучший выбор для современных веб-приложений. Быстрее, удобнее, автоожидания из коробки. Selenium остаётся для legacy-проектов.
